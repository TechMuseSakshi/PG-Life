# 🏠 PGLife – Student Accommodation Finder

PGLife is a web-based student accommodation platform developed using PHP and MySQL. The application helps students search for Paying Guest (PG) accommodations across different cities, view property details, and manage their wishlist through a secure user account.

---

## 📖 Overview

Finding suitable student accommodation can be time-consuming. PGLife simplifies this process by providing an easy-to-use platform where users can browse PG properties, compare facilities, and save their favorite accommodations for future reference.

---

## ✨ Features

- 🔐 User Registration & Login
- 🏙️ Search PGs by City
- 🏠 View Detailed Property Information
- ❤️ Add or Remove Properties from Wishlist
- 👤 User Dashboard
- 📱 Responsive Design
- 🔒 Secure Session Management

## 🚀 Getting Started

### Prerequisites

- XAMPP
- PHP
- MySQL

### Installation

1. Clone this repository

```bash
git clone https://github.com/YOUR_USERNAME/PGLife.git
```

2. Move the project folder to:

```
xampp/htdocs/
```

3. Start **Apache** and **MySQL** using the XAMPP Control Panel.

4. Import the project database into phpMyAdmin.

5. Open your browser and visit:

```
http://localhost/PGLife/
```
